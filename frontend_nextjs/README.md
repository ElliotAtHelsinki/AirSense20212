Project use follow stack : 

- Nextjs
- Antd
- Tailwindcss
- React-query
- Redux


`prepare` : 
```
    yarn install
``` 
`run project`
```
    yarn dev
```

`build image`: 
```
    docker build -t phungvansyhb/airsense-fe .
```

`run image` : 
```
    docker run -itd -p 3000:3000 phungvansyhb/airsense-fe
```
